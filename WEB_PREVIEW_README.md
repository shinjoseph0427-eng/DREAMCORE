# DREAM CORE 웹 프리뷰 실행 안내

이 폴더의 웹 프리뷰는 현재 DREAM CORE 프로토타입의 분위기를 브라우저에서 빠르게 경험하기 위한 가벼운 로컬 미리보기입니다.

최종 Unity 버전이 아니라, Unity Play Mode 테스트 전에 로컬 브라우저에서 복도 분위기와 이동감을 확인하기 위한 프리뷰입니다. 최종 게임 동작은 Unity Play Mode에서 따로 확인해야 합니다.

## 실행 방법

1. `D:\DREAM CORE\PLAY_WEB_PREVIEW.cmd`를 더블클릭합니다.
2. 명령 프롬프트 창이 열리고 아래 주소가 표시됩니다.
3. 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:8777
```

`D:\DREAM CORE\OPEN_WEB_PREVIEW.url` 파일을 더블클릭해도 같은 주소를 열 수 있습니다.

## 조작법

```text
WASD = 이동
Mouse = 시점 조작
Click = 포인터 잠금
ESC = 마우스 잠금 해제
```

## 프리뷰 내용

브라우저 프리뷰에는 DREAM CORE 복도의 핵심 분위기를 담았습니다.

- 아쿠아 블루 타일 복도
- 중앙의 얕은 물길
- 금속 난간
- 벽에 붙은 구름 오브젝트
- 파스텔 옐로우, 시안, 핑크, 퍼플 조명
- 멀리 보이는 빛나는 핑크 문
- 부드러운 안개와 몽환적인 림널 분위기

## 참고

이 프리뷰는 Three.js로 만든 로컬 브라우저용 미리보기입니다. Unity 프로젝트 파일, 생성된 Unity 씬, 게임플레이 스크립트를 대체하지 않습니다.
